Minimal COntrols
================

Mandatory
---------

1. Play/Pause
2. Forward
3. backward
4. Playback Slider with time elapsed and total time
5. Volume Control Slider

Optional
--------

1. Repeat 
2. Shuffle

3. Fav: Get from Ratings
4. Show Media Info
5. Open Button
6. Visualiser
7. Subtitles
8. Tempo
9. Radio
10. Lyrics
11. Last.fm
12. Youtube
13. Sleep Timer
14. UI Splash at start
15. Themeable GUI

Views

2. Grid
3. List
4. Complex list
